C'est quoi un Bundle ?

Présentation

  • Element principal du framework : tout est un bundle
  • Similaire à un plugin ou un module
  • Représente une fonctionnalité
  • Doit être facile à
    • construire
    • distribuer
    • réutiliser

Un bundle contient :

  • des fichiers PHP
  • des fichiers de configuration
  • des fichiers de templates
  • des fichiers CSS
  • des fichier JS
