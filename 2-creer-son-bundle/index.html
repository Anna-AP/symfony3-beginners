Créer son Bundle

C'est quoi un Bundle ?

L'équivalent d'un plugin, sauf que :

  1. Symfony est constitué de bundles
  2. Ils sont donc facile a réutiliser
  3. et facile à construire et distribuer

Un bundle implémente une fonctionalité, il rassemble :

  • les fichiers php
  • les templates
  • CSS
  • JS
  • les tests

L'autoloader

( Deux formats : PSR-0 et PSR-4 )

Règles pour les noms de bundle

  1. CameCase !
  2. 2 mots
  3. Suffixer avec Bundle
  4. ex: FirstBundle

Créer son Bundle

Facile ! 


						php bin/console generate:bundle
