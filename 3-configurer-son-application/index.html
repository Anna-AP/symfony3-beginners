Les Paramètres

parameters.yml

Tout ce qui est spécifique à chaque server

Appelé avec %NOMDUPARAMETRE%

Ex : %secret%

Secret

  • Unique (pour une application)
  • Ajoute de l'entropie à la sécurité
  • Sert pour les cookies
